Layout 布局 ​

通过基础的 24 分栏，迅速简便地创建布局

基础布局 ​

通过 row 和 col 组件，并通过 col 组件的 span 属性我们就可以自由地组合布局。

分栏间隔 ​

行提供 gutter 属性来指定列之间的间距，其默认值为 0。

列偏移 ​

通过制定 col 组件的 offset 属性可以指定分栏偏移的栏数。

vue
<template>
	<s-row :gutter="20">
		<s-col :span="6">
			<div class="grid-content ep-bg-purple " />
		</s-col>
		<s-col :span="6" :offset="6">
			<div class="grid-content ep-bg-purple " />
		</s-col>
	</s-row>
	<s-row :gutter="20">
		<s-col :span="6" :offset="6">
			<div class="grid-content ep-bg-purple " />
		</s-col>
		<s-col :span="6" :offset="6">
			<div class="grid-content ep-bg-purple " />
		</s-col>
	</s-row>
	<s-row :gutter="20">
		<s-col :span="12" :offset="6">
			<div class="grid-content ep-bg-purple " />
		</s-col>
	</s-row>
</template>

Row API ​

Attributes ​

属性名说明类型默认值
gutter栅格间隔number0
tag自定义元素标签stringdiv

Slots ​

插槽名说明子标签
default自定义默认内容Col

Col API ​

Attributes ​

属性名说明类型默认值
span栅格占据的列数number24
offset栅格左侧的间隔格数number0
tag自定义元素标签stringdiv

Slots ​

插槽名说明
default自定义默认内容
